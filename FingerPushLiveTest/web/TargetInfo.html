타겟팅 푸시 정의

타겟팅 푸시는 특정 사용자에게 푸시알림을 보내는 기능입니다. 한 사람이 여러 개의 단말기를 소유한 경우 동일한 식별자로 단말기를 등록하면 다수의 단말기로 푸시 알림이 발송됩니다.


식별자(identity) 정의

사용자를 구분하기 위한 식별자로 예를 들어, 이메일, 아이디, 일련번호 등 개인을 구분하기 위한 값을 설정하면 됩니다.


타겟팅 이용 안내

  • ① 라이브 앱을 스마트폰에 설치
  • ② 라이브 앱을 실행하고 타겟팅 메뉴를 선택
  • ③ 식별자(identity)를 입력하고 등록
  • ④ 실제 서비스 연동시에는 서비스 회원의 아이디 또는 이메일이 식별자로 지정 가능
  • ⑤ 핑거푸시 서비스 접속(www.fingerpush.com)
  • ⑥ 우측 상단 사용자 아이디 클릭 ➔ SERVICE ➔ SEND PUSH ➔ Targetting Push 메뉴에서 발송 가능
