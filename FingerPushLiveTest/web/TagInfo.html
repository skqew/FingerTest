태그 푸시 정의

태그 푸시는 알림을 받는 사용자들을 태그(키워드)로 구분하여 푸시 발송시 그룹별 푸시를 보내는 기능입니다. 예를 들어, 사용자들을 지역별, 연령별, 성별로 구분하여 태그를 설정하면 푸시 발송시 해당 태그를 선택하여 푸시 알림이 가능하고, 다중 태그 선택을 통해 and조건의 알림도 가능합니다.


태그 이용 안내

  • ① 라이브 앱을 스마트폰에 설치
  • ② 라이브 앱 메뉴 중 '태그' 를 선택
  • ③ '모든 태그' 를 선택하면 나를 포함한 모든 사용자 태그 표시
  • ④ '내 태그' 를 선택하면 현재 단말기 태그 표시
  • ⑤ 태그 입력란에 '핑거푸시'라고 입력하고 등록을 선택하면 내 태그 목록에 '핑거푸시' 태그 확인 가능
  • ⑥ 태그 푸시를 발송하려면 www.fingerpush.com에서 테스트 계정(partner@kissoft.co.kr, partner1234)으로 로그인 한 후 우측 상단의 아이디를 클릭하고 'Service' 메뉴를 선택
  • ⑦ 좌측 메뉴 중 'Send Push'를 클릭하고 App 을 선택한 후 메세지를 입력
  • ⑧ 'Settings' 에서 원하는 'Tag' 를 선택하고 푸시를 발송
